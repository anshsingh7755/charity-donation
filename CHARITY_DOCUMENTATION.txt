RESPONSIVE DESIGN DETAILS
================================================================================

CSS CLAMP() FUNCTION USAGE

The website uses CSS clamp() for fluid, responsive sizing without media queries:

Logo Font Size: clamp(16px, 5vw, 24px)
  - Minimum: 16px on tiny screens
  - Scales with 5% of viewport width
  - Maximum: 24px on large screens
  - Automatically adjusts between min and max

Logo Gap: clamp(5px, 2vw, 10px)
  - Spacing automatically adjusts
  - No media query needed

Navbar Padding: clamp(10px, 2vw, 20px)
  - Padding scales with viewport
  - Responsive on all screen sizes

Navigation Gap: clamp(15px, 3vw, 30px)
  - Menu item spacing adjusts automatically
  - Optimal spacing on all devices

Icon Sizing: clamp(40px, 10vw, 48px)
  - Icons scale with screen size
  - Always readable and clickable

MEDIA QUERIES

@media (max-width: 768px)
- Hamburger menu appears
- Navigation layout changes
- Text sizes reduce appropriately
- Grids convert to single column
- Padding and margins adjust

@media (max-width: 480px)
- Further text size reductions
- Buttons optimize for touch
- Grid items stack vertically
- Form fields full width
- Hero section reduces padding

LAYOUT SYSTEMS

Flexbox Usage
- Navigation bar layout
- Footer content arrangement
- Form field alignment
- Button positioning
- Menu item spacing

CSS Grid Usage
- Causes card grid
- Quotes section grid
- Statistics grid
- Footer content grid
- Responsive grid with auto-fit

================================================================================
ANIMATION SPECIFICATIONS
================================================================================

CSS ANIMATIONS

fadeInUp Animation
@keyframes fadeInUp
  From: opacity 0, translateY 30px
  To: opacity 1, translateY 0
  Duration: 0.6s ease
  Used on: Cards, quotes, statistics

slide Animation
@keyframes slide
  From: translateX 0
  To: translateX 40px
  Duration: 20s linear infinite
  Used on: Hero background pattern

progressFill Animation
@keyframes progressFill
  From: width 0
  To: width actual-percentage
  Duration: 1.5s ease
  Used on: Progress bars

slideUp Animation
@keyframes slideUp
  From: translateY 50px, opacity 0
  To: translateY 0, opacity 1
  Duration: 0.3s ease
  Used on: Modal popup

Intersection Observer
- Triggers animations when elements enter viewport
- Efficient (not every scroll event)
- Used for cards, quotes, stats, progress bars
- Only animates once per element
- Improves performance
