CORE CODE SNIPPETS
================================================================================

KEY HTML STRUCTURE:
  <header>Navigation and Logo</header>
  <section class="hero">Hero Section</section>
  <section class="causes">Donation Causes</section>
  <section class="donation">Donation Form</section>
  <footer>Footer</footer>

KEY CSS FEATURES:
  - Mobile-first responsive design (breakpoints: 320px, 481px, 769px, 1201px+)
  - Gradient backgrounds for modern look
  - Smooth transitions and hover effects
  - Flex and Grid layouts for responsive design
  - Animation effects (fade-in, slide-up) using CSS animations

KEY JAVASCRIPT FUNCTIONS:
  - validateForm() → Check donation form fields
  - validateContactForm() → Check contact form fields
  - smoothScroll() → Smooth page navigation
  - Intersection Observer API → Trigger animations on scroll
  - Form submission handlers

PYTHON PDF GENERATION:
  - Use ReportLab library
  - Create A4-sized PDF with 1-inch margins
  - Add text, tables, and images
  - Embed screenshots of website
  - Generate professional report document

HOW TO RUN THE PROJECT
================================================================================

STEP 1: SETUP (First time only)
  - Open terminal in project folder
  - Create virtual environment: python -m venv .venv
  - Activate it: .venv\Scripts\activate
  - Install ReportLab: pip install reportlab

STEP 2: RUN WEBSITE
  - Open index.html in any web browser
  - Or use Live Server extension in VS Code
  - Website will load with all features

STEP 3: GENERATE PDF
  - Run: python generate_pdf.py
  - This creates NGO_Report.pdf
  - Report includes all documentation and screenshots

STEP 4: PUSH TO GITHUB
  - git add .
  - git commit -m "Final charity website"
  - git push origin main
